Welcome

Doing what I love.

I am a Full-Stack Developer who is design-minded. Most of my work has been with JavaScript. Although I am currently in love with the M.E.R.N. stack, I always enjoy learning and working with new technologies. You can view my resume here, and you can view some of my latest projects down below.

Outcache

A peer-to-peer rental website for outdoor equipment. Built using the MERN stack. I was responsible for the front-end of this website, as well as some back end routing and component work.

Mongo News Scraper

A Node.js app that uses Cheerio.js to scrape the web (FinViz.com) for news articles, and stores them in a MongoDB database. Links to those articles are then pulled from the database and displayed on the homepage, and users can also create/delete comments on each article. This app uses mLab for hosting the database.

SQL burger app

A simple Node.js & MySQL app hosted on Heroku, which allows users to add burgers to the list and 'devour' them. This app uses JawsDB for hosting the database.

Gif Search

A simple gif search tool using the Giphy API. This was made using jQuery and Bootstrap, and uses local storage to save search terms.

Card Fighting Game

A simplistic card-fighting game where the user must choose a hero and defeat all enemies to win. I am most proud of the animations I created while building this app. This was my first time trying animations.
